# 対戦マッチアップアニメーション
GitHub Pages 上で動作する対戦マッチアップアニメーション生成ツール。

## 🚀 使い方
1. 画像をアップロード
2. テキストを入力
3. 「プレビュー」ボタンでアニメーション再生
4. 「GIFダウンロード」ボタンでGIFを保存

## 📂 ファイル構成
- `index.html` - メインページ
- `script.js` - アニメーション＆GIF生成スクリプト
- `style.css` - スタイル設定
- `gif.worker.js` - GIF作成用スクリプト
- `images/` - 画像フォルダ
